### GLOBAL HEALTH STATISTICS: FROM DATA CLEANING TO INTERACTIVE DASHBOARD



#### PROJECT OVERVIEW



This project focuses on the end-to-end processing of global health data, cleaning it for accuracy and building an interactive dashboard to visualize key health metrics.



##### 1\. DATA CLEANING PHASE



I performed the following steps to ensure data integrity:



&nbsp; Duplicate Removal: Used Excel to identify and remove redundant records ensuring that each health case was only counted once.



&nbsp; Standardization: Cleaned and Standardized country names and disease categories.



&nbsp; Data Formatting: Standardized numerical values for Prevalence, Incidence and Mortality Rates to allow for accurate averaging and comparison in the dashboard.



##### 2\. INTERACTIVE DASHBOARD 



The final dashboard provides a high level view of global health trends through three main visualizations:



Mortality Rates By Country(Bar Chart): Compares the average death rates across various nations to identify regional health challenges.



Mortality Rates By Age and Gender(Grouped Bar Chart): Breaks down deaths by gender(Male and Female) and age brackets(0-18, 19-35,etc.)



Incidence Rate By Country(Line Graph): Tracks the average rate of new diseases cases reported per country.



##### 3\. DYNAMIC EXPLORATION(SLICERS)



To make the dashboard truly interactive, I implemented three slicers that filter all charts simultaneously:



Year: To view data trends specifically between 2000 and 2005.



Disease Category: To filter the view to specific conditions such as Alzheimer's, Asthma, Cancer, Cholera or COVID-19.



Doctors Available: Allows users to filter by the number of medical professions available per 1000 people to see if better healthcare correlates with lower mortality rates.





TOOLS USED

Data Cleaning: Excel.



Data Visualizations: Excel Pivot Tables and Charts.
